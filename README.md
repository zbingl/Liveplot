# Real-Time Data Plotter with SQLite

## Overview

Liveplot visualizes data points from an SQLite database in real-time, and displays them on a graph in your web-browser.

## Features

- Real-time graph updates
- SQLite backend for storing time-series data
- Flask web interface to display the graph
- Random data generation for testing

## Technologies Used

- Python 3.x
- SQLite
- Flask
- matplotlib
- NumPy
- Watchdog
